    13  // LogsArithmeticProcessor Use the Arithmetic Processor to add a new attribute (without spaces or special characters
    14  // in the new attribute name) to a log with the result of the provided formula.
    15  // This enables you to remap different time attributes with different units into a single attribute,
    16  // or to compute operations on attributes within the same log.
    17  //
    18  // The formula can use parentheses and the basic arithmetic operators `-`, `+`, `*`, `/`.
    19  //
    20  // By default, the calculation is skipped if an attribute is missing.
    21  // Select “Replace missing attribute by 0” to automatically populate
    22  // missing attribute values with 0 to ensure that the calculation is done.
    23  // An attribute is missing if it is not found in the log attributes,
    24  // or if it cannot be converted to a number.
    25  //
    26  // *Notes*:
    27  //
    28  //   - The operator `-` needs to be space split in the formula as it can also be contained in attribute names.
    29  //   - If the target attribute already exists, it is overwritten by the result of the formula.
    30  //   - Results are rounded up to the 9th decimal. For example, if the result of the formula is `0.1234567891`,
    31  //     the actual value stored for the attribute is `0.123456789`.
    32  //   - If you need to scale a unit of measure,
    33  //     see [Scale Filter](https://docs.datadoghq.com/logs/log_configuration/parsing/?tab=filter#matcher-and-filter).
    34  type LogsArithmeticProcessor struct {
    35  	// Arithmetic operation between one or more log attributes.
    36  	Expression string `json:"expression"`
    37  	// Whether or not the processor is enabled.
    38  	IsEnabled *bool `json:"is_enabled,omitempty"`
    39  	// If `true`, it replaces all missing attributes of expression by `0`, `false`
    40  	// skip the operation if an attribute is missing.
    41  	IsReplaceMissing *bool `json:"is_replace_missing,omitempty"`
    42  	// Name of the processor.
    43  	Name *string `json:"name,omitempty"`
    44  	// Name of the attribute that contains the result of the arithmetic operation.
    45  	Target string `json:"target"`
    46  	// Type of logs arithmetic processor.
