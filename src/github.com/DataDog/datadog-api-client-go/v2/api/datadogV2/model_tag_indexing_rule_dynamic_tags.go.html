    11  // TagIndexingRuleDynamicTags Options for dynamic tag indexing applied per metric, such as tags filtered by query usage.
    12  //
    13  // Before a tag key is dropped by this rule, two grace period conditions must be met:
    14  //
    15  // 1. The metric must be submitted for at least as long as the selected window.
    16  // 2. A tag key must have been submitted for at least 15 days.
    17  //
    18  // Any metric or tag key that does not meet these conditions are excluded from this
    19  // indexing rule. The `exclude_not_*` fields require `exclude_tags_mode` to be set to `true`.
    20  type TagIndexingRuleDynamicTags struct {
